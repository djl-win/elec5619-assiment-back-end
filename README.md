This project is a visualisation platform implemented by Springboot + Hibernate + Mysql. Here are some details of the project.  


### Project information:  
1. JDK version: 1.8  
2. Springboot Version: 2.7.3  
3. Database: Mysql 5.7.38  
4. Maven: 3.6.3  
5. Port: 8080  

### Step to deploy the environment:    
1. Maven install  
2. Maven clear  
3. Maven package
